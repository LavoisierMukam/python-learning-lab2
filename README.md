# Lab 2 — Data Visualization and Time Series Analysis

This repository contains my submission for **Lab 2** of the Python course.

## Contents

- `lab2_Lavoisier_MUKAM_github.ipynb`: main notebook
- `nyc_taxi.csv`: NYC taxi dataset used for the Plotly part
- `TG_STAID011249.txt`: temperature dataset used in the final section

## Topics covered

- plotting with `matplotlib`
- exporting figures in `.png` and `.pdf`
- dual-axis plots with `twinx()`
- subplot layouts
- interactive visualization with `plotly`
- grouping and aggregation with `pandas`
- time series analysis with `DateTimeIndex`
- comparison between electricity consumption and temperature

## Notes

Some sections assume that the required datasets are available locally in the same working directory as the notebook.

For the temperature dataset, the number of header lines may vary depending on the downloaded file version. If needed, the `skiprows` parameter in the notebook can be adjusted.
